# Prasham Parekh's Portfolio

Welcome to my personal portfolio website! This site showcases my projects, skills, and experiences as a Computer Science graduate passionate about software development, AI, and creative problem-solving.

🚀 **Live Website**
- [prasham-parekh.github.io/portfolio](https://prasham-parekh.github.io/portfolio/)

📄 **About**

This portfolio highlights:
- My background, education, and interests in technology
- A curated collection of projects with interactive detail views
- A full HTML resume with PDF download
- A contact form for collaborations or opportunities

💻 **Technologies Used**
- HTML5 for structure
- CSS3 with custom properties for styling and theming
- JavaScript for interactivity, dark mode, and project modals
- Google Fonts (Roboto Condensed) for typography

✨ **Features**
- **Dark Mode** — toggleable light/dark theme saved to localStorage, synced across all pages
- **Projects Carousel** — horizontal scrolling cards that open into a browser-style modal with project details, tech stack, and GitHub links
- **Responsive Design** — optimized for phones, tablets, laptops, and ultrawide monitors (360px to 2560px+)
- **HTML Resume** — styled resume page with a download PDF button, fully themed with dark mode
- **Smooth Navigation** — active section highlighting on scroll, smooth scrolling between sections
- **Contact Form** — form UI for reaching out (backend functionality coming soon)

📦 **How to Run Locally**

1. Clone the repository:
   ```
   git clone https://github.com/prasham-parekh/portfolio.git
   ```
2. Navigate to the project folder:
   ```
   cd portfolio
   ```
3. Open `index.html` in your browser.
